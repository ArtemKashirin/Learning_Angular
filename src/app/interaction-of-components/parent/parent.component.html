Компонентное взаимодействие - сценарии взаимодействия компонентов, в которых два или более компонента совместно используют информацию:

  • 1. Передача данных от родителя к потомку с привязкой ввода
  • 2. Передача данных от родителя к потомку, перехват и изменение входных свойств с помощью Get и Set
  • 3. Перехватите изменения свойств ввода с помощью ngOnChanges() в потомке
  • 4. Родитель прослушивает дочернее событие
  • 5. Родитель взаимодействует с дочерним элементом, используя локальную(ссылочную) переменную
  • 6. Родитель вызывает потомка при помощи функции с декаратором @ViewChild()
  • 7. Родитель и потомок(-и) общаются с помощью сервиса
